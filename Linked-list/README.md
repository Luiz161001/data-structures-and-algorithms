# Linked List 🔗

This section focuses on building and understanding linked lists from scratch, followed by applying them to solve common problems.

## Overview

I started by implementing a **doubly linked list** to get a solid understanding of how nodes connect in both directions and how operations affect memory and pointers.

From here, I’m solving problems to reinforce these concepts and improve problem-solving speed.

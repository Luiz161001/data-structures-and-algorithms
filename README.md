# Data Structures & Algorithms 🚀

This repository is my personal space to deeply understand data structures and algorithms by building everything from scratch and applying them to real problems.

Instead of just solving problems, I focus on implementing each data structure first, then using it to solve challenges. The goal is to truly understand how things work under the hood.

## Structure 📁

Each data structure is organized into two parts:

- implementation → building the data structure from scratch ⚙️  
- problems → solving problems (mostly LeetCode-style) using that structure 🧩  

Example:

linkedlist/
  implementation/
  problems/

trees/
  implementation/
  problems/

## Topics Covered 🧠

- Linked Lists
- Stacks & Queues (in progress)
- Trees & Binary Search Trees (in progress)
- Graphs (in progress)
- Recursion & Backtracking (in progress)
- Sorting & Searching (in progress)
- Dynamic Programming (in progress)

## Why this repo? 🎯

I wanted to go beyond just solving problems and actually understand how things work under the hood.

For each topic, I start by building the data structure myself, then use it to solve problems. This way I can connect the theory with real use cases.

## Tech 💻

- C++
- Focus on time & space complexity
- Clean and readable implementations
